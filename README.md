# HR-Analytics_Dashboard
Power Bi Dashboard
project is an HR Analytics Dashboard developed using Power BI to analyze employee attrition and workforce metrics.
The dashboard provides actionable insights into attrition rate, salary slabs, job roles, education, gender, tenure, and departments, helping HR teams make data-driven retention decisions.
This project demonstrates strong skills in data analysis, KPI design, DAX, and business intelligence reporting.
Project Objectives:
Analyze employee attrition trends,
Identify key factors influencing employee turnover,
Compare attrition by salary, job role, education, gender, and tenure,
Track overall workforce demographics,
Support HR decision-making and workforce planning.
